# Página 1 – ¿Qué es el Modelo de Von Neumann?

El **Modelo de Von Neumann** es una arquitectura de computadora propuesta por John von Neumann en 1945.  
Su idea principal es que **los datos y las instrucciones se almacenan en la misma memoria**, algo que hoy damos por normal pero que en su época fue una innovación enorme.

## ¿Por qué fue importante?

Antes de este modelo, los ordenadores funcionaban conectando y desconectando cables para programarlos. Von Neumann propuso que las instrucciones estuvieran dentro de la memoria, igual que los datos, lo que permitió:

- Tener **programas almacenados**  
- Ejecutar instrucciones de manera **secuencial**  
- Crear máquinas más rápidas, flexibles y programables  

## Elementos básicos del modelo

- **Unidad de Control** – interpreta instrucciones  
- **ALU** – realiza operaciones matemáticas y lógicas  
- **Memoria** – guarda datos e instrucciones  
- **Buses** – permiten la comunicación interna  

Este modelo es la base de casi todos los ordenadores actuales.
